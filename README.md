# hass-wifigate
Component to integrate Home Assistant with WifiGate controller.

## Installation
[![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)](https://github.com/hacs/integration)

1. Open HACS Settings and add this repository (https://github.com/valkuc/hass-wifigate/)
   as a Custom Repository (use **Integration** as the category).
2. The `WifiGate` page should automatically load (or find it in the HACS Store).
3. Click `Install`.
